**`bykcli`** 是一个轻量的命令行工具集合 🧰

用于通过简单命令解决一些日常的小需求。

比如：
* 没有现成工具可用
* 或者工具太重、不够直接
* 想一条命令快速搞定

项目采用插件化结构，子命令以插件形式组织，支持按需插拔

## 快速开始

- 使用 pip 安装

```bash
pip install bykcli
```

- 显示仪表盘

```bash
byk
```
