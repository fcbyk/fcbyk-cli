## Slide 子命令

Slide 用于在局域网内快速搭建一个 PPT 远程控制服务，通过手机等移动设备打开网页即可控制本机正在播放的幻灯片，并支持基础的鼠标控制。

### 基本功能
- 启动一个基于 Web 的 PPT 远程控制服务
- 通过访问同一局域网内的控制页面，进行上一页/下一页/跳转首尾页操作
- 支持通过网页控制鼠标移动、单击、右键、长按、滚轮滚动等操作
- 通过访问密码保护控制页面，避免他人误触

### 基本用法
```bash
fcbyk slide [选项]
```

启动后，命令会：
- 检查端口是否可用
- 交互式询问访问密码，并要求确认（不能为空）
- 创建并启动基于 Flask + Socket.IO 的 Web 控制服务
- 输出当前局域网可访问的地址列表
- 自动将其中一个访问地址复制到剪贴板
- 使用该地址在手机浏览器中打开控制页面

### 参数说明
- `-p, --port INTEGER`  
  指定 Web 服务端口，默认 `80`。  
  例如：`-p 8080`

### 常见用法示例

1. 使用默认端口启动 Slide 服务（端口 80）
```bash
fcbyk slide
```

2. 在自定义端口启动 Slide 服务
```bash
fcbyk slide -p 8080
```

3. 在会议现场使用 Slide 控制 PPT 的典型流程
```bash
# 1. 在演示电脑上启动 Slide 服务
fcbyk slide -p 8080

# 2. 在同一 Wi-Fi 下，用手机扫码/打开控制地址
# 3. 输入刚才设置的访问密码完成登录
# 4. 使用手机页面进行翻页和鼠标控制
```

### 注意事项
- Slide 通过模拟键盘和鼠标事件工作，请确保运行环境允许此类操作，并避免与其他自动化软件冲突。
- 请在演示前先本地测试一次，以确保当前系统、PPT 软件与 pyautogui 配合正常。
- 为避免误操作，建议只在需要时启动 Slide，并妥善保管访问密码。
- Slide 仅面向局域网场景设计，不推荐直接暴露到公网环境。
